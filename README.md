# FPLInsights

FPLInsights is a Python project for simulating and predicting Fantasy Premier League scores using Monte Carlo methods.

## Features
- Local SQLite data storage
- Update pipeline from the FPL API
- Basic team prediction model
- Advanced fixture-aware prediction model
- Monte Carlo simulation

## Documentation
All documentation is in the `docs/` directory:
- basic.md
- advanced.md
- overview.md
- roadmap.md
- architecture.md
- api.md

## Usage
Run the update pipeline, then call prediction modules or test scripts.

FPLInsights is designed for future extensions including Poisson modeling, clean sheet models, autosubs logic, and AI-assisted prediction commentary.
